##使用说明
![](pic/intro.PNG)
1. 选择ifun里面的兼容模式
2. 打开各个浏览器的开发者界面devtools
3. 选择network
4. 选择一个以media 开头的文件
5. 右边选择headers
6. 拷贝下面的url备用

![](pic/ablauf.PNG)
##
7. 依次输入url和想要的名字
8. 在out1目录下就有了想要的视频
